# Рекламный баннер главной страницы на всю ширину контейнера

## Контекст

На главной странице рекламный блок был дополнительно ограничен шириной `970px`, из-за чего на десктопе оставались заметные боковые поля внутри основного контейнера страницы.

## Что сделано

- Убран локальный лимит ширины у заголовка рекламного блока.
- Убран локальный лимит ширины у баннера.
- Для десктопа задана пропорциональная высота баннера, чтобы при расширении изображение не обрезалось слишком сильно.
- Мобильная высота оставлена через существующий адаптивный `clamp(...)`.

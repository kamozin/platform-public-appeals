# Сокращение формы обращения до 4 шагов

## Context

Страница подачи обращения в Nuxt была разбита на 7 шагов: категория, описание, файлы, место, контакты, SmartCaptcha и проверка. Для более короткого пользовательского сценария форму нужно привести к 4 шагам, соответствующим промо-блоку на главной странице.

## Goal

Сократить мастер подачи обращения до 4 шагов без изменения API-контракта и без удаления существующих полей формы.

## Non-goals

- Не менять backend API.
- Не менять формат черновика обращения.
- Не менять `static/`.
- Не внедрять реальную интеграцию SmartCaptcha в этой задаче.

## Backend changes

Backend не изменяется.

## Frontend changes

- Обновлён список шагов мастера:
  1. Тема.
  2. Проблема.
  3. Доказательства.
  4. Отправка.
- Блок адреса и ориентира перенесён в шаг описания проблемы.
- Контакты, SmartCaptcha, проверка данных и отправка объединены в финальный шаг.
- Переходы в блоке проверки перенастроены на 4-шаговую структуру.
- CSS прогресса формы изменён с 7 на 4 колонки.

## API contract

API-контракт не меняется. Используются прежние endpoints черновика и отправки обращения.

## SEO impact

Страница формы остаётся `noindex`; SEO-изменений нет.

## Tests

Выполнено:

- `rg`-проверка отсутствия ссылок на шаги 5-7 в форме.
- `git diff --check` для изменённых файлов.

Не выполнено:

- Полный `pnpm typecheck`: текущая Windows-среда использует Node 20.16, а `pnpm@11.7.0` требует Node 22.13+. WSL содержит Node 12 и не подходит для Nuxt 4. Запуск pnpm из Windows пытался пересоздать WSL `node_modules`, поэтому он был остановлен.

## Acceptance criteria

- В прогрессе формы отображаются 4 шага.
- Пользователь может выбрать тему на шаге 1.
- Пользователь может заполнить описание, срочность и место на шаге 2.
- Пользователь может приложить файлы на шаге 3.
- Пользователь может указать контакты, пройти SmartCaptcha, проверить данные и отправить обращение на шаге 4.
- Отправка использует существующий API черновика.

## Postman

Postman collection не требуется обновлять, потому что API endpoints не менялись.
